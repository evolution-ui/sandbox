> A playground for the new Evolution UI directory structure and workflow.

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

Navigate to `http://localhost:3000/[standard|evolution]/[component_name]/[variant_name].html` to view a component variant in the browser.
